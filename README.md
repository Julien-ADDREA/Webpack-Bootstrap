<h1 align="center">Quick start</h1>

1. Clone repository :
```bash
git clone https://github.com/Julien-ADDREA/Webpack-Bootstrap
```
2. Install dependencies :
```bash
npm install
```
3. Build your first app :
```bash
npm run build:development
npm run build:production
```
